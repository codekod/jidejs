This demo shows how to use a {@link jidejs/ui/control/Tooltip} with a {@link jidejs/ui/control/Button}. Tooltips can be added to any controls using the same API.

require([
    'jidejs/ui/control/Tooltip', 'jidejs/ui/control/Button', 'jidejs/ui/control/HTMLView'
], function(Tooltip, Button, HTMLView) {
    var showTooltip = new Button({
        text: 'Hover to see the tooltip.',
        tooltip: new Tooltip({
            content: new HTMLView({
                content: 'This is the tooltip. It can contain any content you want.'
            })
        })
    });

    // add the 'element' of the ListView to the DOM
    root.appendChild(showTooltip.element);
});

To specify a {@link jidejs/ui/control/Tooltip} that should be displayed when the user moves the mouse over a control, just assign it to the {@link jidejs/ui/Control~tooltip} property.

Every {@link jidejs/ui/Control} has that property and knows how to display a tooltip.

Important Tooltips should not be the only way to discover the meaning or usage of a control since they can't be implemented for touch interfaces such as most mobile devices.

This demo shows how to style jide.js using simple CSS.

It uses the Lion CSS UI Kit for the styling of the components and requires a custom license that is not included in jide.js.

According to the author, the design is optimized for usage in WebKit based browsers. We did not take any special measures to make sure it works other browsers (i.e. IE, Firefox, Opera).

Enter text into the text fields and the text area.

Click the Send button to see a preview of your email.

Click on one of the section titles to see them collapse or expand.
